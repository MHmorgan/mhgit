MHgit is a simple git library for interracting with git repositories. Provides an idiomatic and easy
way of dealing with git repos.

Requires git to be installed on the system.

#### Supported actions

* `add`
* `clone`
* `commit`
* `init`
* `notes`
* `pull`
* `push`
* `remote`
* `status`
* `stash`
* `tag`
